为什么选择 Tailwind CSS？

1. 直观易维护

  • 所见即所得：bg-player-1 直接对应玩家 1 的颜色
  • 快速修改：改样式只需改 className，不需要跳转到 CSS 文件
  • 无命名烦恼：不需要想 CSS 类名，用 Tailwind utility classes

2. 性能优化

  • 按需生成：只生成用到的样式，最终 CSS 只有 10-30KB
  • 无重复代码：复用 utility classes，避免 CSS 冗余
  • Tree-shaking：生产环境自动去除未使用的样式

3. 完美适配游戏 UI

  • 动态颜色：bg-[${playerColor}] 可动态生成玩家颜色
  • 响应式：md:grid-cols-12 轻松实现移动端适配
  • Hover 效果：hover:scale-110 地图单元格交互一行搞定

4. 对比 MUI

对比项 Tailwind CSS MUI
Bundle Size ✓ 10-30KB ✗ 300KB+
自定义程度 ✓ 完全自由 △ 需覆盖默认样式
学习曲线 ✓ 1-2天上手 △ 需理解 Material Design
游戏 UI 适配 ✓ 完美 ✗ 风格冲突
