设计原则总结

1. 双色系统

  • 主界面：灰色系统（#FAFAFA - #2A2A2A），营造专业、专注氛围
  • 游戏内：8 色玩家系统，高对比度，易于区分
  • 过渡自然：从主界面进入游戏，视觉过渡流畅

2. 清晰线条

  • 无纹理：所有线条清晰锐利，没有铅笔画纹理
  • 对眼睛友好：长时间游戏不疲劳
  • SVG 优先：所有图标使用 SVG，可缩放且清晰

3. 圆角设计

  • 柔和亲切：6px 圆角使界面更友好
  • 现代感：符合现代 UI 设计趋势

4. 色盲友好

  • 高对比度：8 种玩家颜色都具有足够的对比度
  • 多种区分方式：除了颜色，还有图标（♔◆▲）辅助识别
